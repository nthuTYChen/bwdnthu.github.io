Introduction

Welcome to Bilingual Web Design offered in Fall 2020! Here's the introduction to this course to help you get ready.

Why this course?

  1. First of all, you probably want to learn some practical skills.
  2. Second, you need a Webpage to present your profile/works/personality.
  3. Third, Web Design helps cultivate your logical thinking and creativity.
  4. Finally, you still need to practice your English writing skills.

What to learn?

HTML5, CSS, JavaScript(!!!)

HTML5 and CSS are probably more entertaining since you get to see what you create immediately. JavaScript is perhaps a bit more intimidating - After all, computer programming sometimes means a steep learning curve.

But that's exactly why you're here, to step out of your comfort zone.

What do you need?

  1. A GitHub account to host your Webpage (see Preparation below). You can use the Web space provided by NTHU, but the Web address may look a bit dumb (for example: http://oz.nthu.edu.tw/~g10xxxxxx).
